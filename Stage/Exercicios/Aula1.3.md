# O Padrão Model-View-Controller

Assinale a Alternativa correta:

a.

Views são classes que representam os dados da aplicação. Essas classes usam lógica de validação para reforçar regras de negócios para esses dados.

b.

Models são os componentes que exibem a interface de usuário da aplicação. Em geral essas interfaces exibem os dados da view.

c.

Controllers são classes que tratam as requisições do navegador. Elas obtêm dados do modelo e invocam templates de views que retornam uma resposta.

d.

Controllers são os componentes que exibem a interface de usuário da aplicação. Em geral essas interfaces exibem os dados da view.

> Opinião da Alura:
> Models são classes que representam os dados da aplicação. Essas classes usam lógica de validação para reforçar regras de negócios para esses dados.
> Views são os componentes que exibem a interface de usuário da aplicação. Em geral essas interfaces exibem os dados do modelo.
> Controllers são classes que tratam as requisições do navegador. Elas obtêm dados do modelo e invocam templates de views que retornam uma resposta.
