# Criando uma Solução com Linha de Comando

Você precisa criar uma nova solução utilizando o template do ASP.NET Core MVC, mas em vez de utilizar o Visual Studio, 
você quer editar seus arquivos com o Notepad.

Escolha a alternativa com a linha de comando adequada para criação da solução desejada:

a.
```
c:\projetos\CasaDoCodigo> dotnet new sln
```

b. 
```
c:\projetos\CasaDoCodigo> dotnet new web
```

c. 
```
c:\projetos\CasaDoCodigo> dotnet new mvc
```

d. 
```
c:\projetos\CasaDoCodigo> dotnet new web
```
